# Pi-NFT 评估

这是一个基于 Pi Network 测试网的 NFT 展示项目。

## 项目介绍
- 这是我的第一个 Pi Network NFT Demo
- 包含 NFT 图片、前端展示页面和元数据配置
- 用于学习 Pi 生态 NFT 开发流程

## 访问地址
- 项目首页：https://haojiahuo1120.github.io/pi-nft-assests/
- NFT 元数据：https://haojiahuo1120.github.io/pi-nft-assests/metadata.json

## 技术说明
- 纯前端静态页面（HTML + JSON）
- 图片和元数据托管在 GitHub Pages
- 符合 Pi Network NFT 元数据标准
